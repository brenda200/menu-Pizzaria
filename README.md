# 🍕 Menu Pizzaria

Um cardápio digital simples e elegante para uma pizzaria. O projeto apresenta os sabores, preços e estilo visual de forma atrativa e responsiva, ideal para uso em sites de restaurantes ou como vitrine online.

🔗 [Acesse o menu online](https://brenda200.github.io/menu-Pizzaria/)

## ✨ Funcionalidades

- 📋 Apresentação dos sabores de pizza
- 💰 Preços organizados por categoria
- 📱 Design responsivo para dispositivos móveis
- 🎨 Visual limpo e atrativo

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
